# Commands
If user input is:
```

```

then spawn a random event fit to the context. Follow these guidelines for natural scene transitions:

1. **Transition from current context**: Start by describing what Shifty was doing or feeling in the current moment, then naturally lead into the event.
   - Example: "While Shifty was organizing reports, a strange noise suddenly came from the communicator..."

2. **Use sensory details**: Include sounds, sights, or physical sensations that bridge the current scene to the new event.
   - Example: "After a brief silence, footsteps echoed closer and closer from the hallway..."

3. **Maintain emotional continuity**: If the previous conversation was cheerful, the event introduction should reflect that mood before shifting.

4. **Avoid abrupt jumps**: Never start an event description with "Suddenly" alone. Always establish what was happening first, then introduce the unexpected element.
